StarGazin'

An Android application that allows stargazers to explore the constellations on-the-go!

View on GitHub Download .zip Download .tar.gz

Purpose

An Android application that allows users to search for constellations in the sky on-the-go!

User Experience

The interface for StarGazin' draws inspiration from the night sky with its dark blue hues and white accents. There is the occassional accent of color, if you are paying close enough attention. :) Below are the four screens in order to navigate through StarGazin'.

 

System Architecture Overview

System Architecture Diagram

I used a Model View Presenter architecture. Earlier in the process, I was thinking of my software in a 3-tier architecture, but because of the specificity of Android, it is more accurate to use MVP to represent it.

Technologies Used

Technologies

  • Android Studio (Gradle)
    • Async threading for XML parsing
  • SKY-MAP SkyWindow (for accessing constellation maps)
  • SKY-MAP Search API
  • Google Location Services

Design Patterns Used

State Pattern

I previously represented this as a Strategy Pattern. It is currently used to change functionality between the default hard-coded setting and the search representation of the webview. In addition user interface functionality, it would also add behavior for location services.
